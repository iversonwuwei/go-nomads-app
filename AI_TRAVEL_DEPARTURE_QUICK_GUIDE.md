# AI Travel Planner 出发地功能 - 快速指南

## 🎯 功能简介

在 AI Travel Planner 中添加出发地信息，让旅行计划更精准！

## 📍 使用方法

### 方式 1：手动输入出发地

1. 打开城市详情页
2. 点击 **AI Travel Planner** 按钮
3. 在 **Departure Location** 字段输入您的城市名称
   - 例如：`Shanghai`, `Beijing`, `New York`
4. 设置其他旅行参数（天数、预算等）
5. 点击 **Generate AI Plan**

### 方式 2：使用当前位置

1. 打开 AI Travel Planner 对话框
2. 点击出发地输入框右侧的 **📍 定位按钮**
3. 首次使用时允许位置权限
4. 系统自动填充您的当前位置坐标
5. 继续设置其他参数并生成计划

### 方式 3：留空（可选）

- 出发地是可选字段
- 不填写也能正常生成旅行计划
- 但填写后计划会更精准

## 🔧 技术细节

### 位置权限处理

**首次使用：**
- 系统会请求位置权限
- 请选择"允许"以使用定位功能

**权限被拒绝：**
- 仍可手动输入出发地
- 或前往系统设置中开启权限

### 位置格式

**手动输入：** 城市名称（任意格式）
```
Shanghai
上海
New York City
```

**自动定位：** 坐标格式
```
Current Location (31.23, 121.47)
```

## ⚠️ 注意事项

1. **位置权限**
   - 定位功能需要位置权限
   - 手动输入不需要权限

2. **网络连接**
   - 获取位置需要网络连接
   - 离线模式请手动输入

3. **精确度**
   - 手动输入更灵活
   - 自动定位更精确

4. **隐私保护**
   - 位置信息仅用于计划生成
   - 不会上传或保存

## ❓ 常见问题

**Q: 必须填写出发地吗？**
A: 不是必须的，但填写后计划会更精准。

**Q: 定位按钮没反应？**
A: 检查位置权限和网络连接。

**Q: 可以输入中文地名吗？**
A: 可以，支持任意语言的城市名称。

**Q: 位置不准确怎么办？**
A: 使用手动输入功能，更灵活准确。

**Q: 出发地信息会保存吗？**
A: 当前不保存，未来会支持历史记录。
