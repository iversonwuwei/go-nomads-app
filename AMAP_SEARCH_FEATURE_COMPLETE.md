## 📋 功能概述

为高德地图选择器添加了**完整的地址搜索功能**,用户现在可以通过三种方式选择位置:

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| 🔍 **搜索** | 输入地址/POI名称 | 快速定位已知地点 |
| 👆 **点击** | 点击地图任意位置 | 选择可见范围内的位置 |
| 👋 **拖动** | 拖动地图浏览 | 精细调整位置 |

### 核心实现

#### POI搜索
```kotlin
private fun performSearch(query: String) {
    val city = currentCityCode.ifEmpty { 
        currentCity.ifEmpty { "全国" } 
    }
    
    val poiQuery = PoiSearch.Query(query, "", city)
    poiQuery.pageSize = 20
    
    poiSearch.query = poiQuery
    poiSearch.searchPOIAsyn()
}
```
