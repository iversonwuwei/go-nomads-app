# 聊天头像点击查看成员 - 快速指南

## ✅ 功能已完成

在聊天页面中点击对话记录的头像，即可查看该成员的详细信息！

## 🎯 使用方法

### 方式 1: 点击消息头像
```
1. 进入任意城市聊天室
2. 浏览聊天记录
3. 点击他人消息旁的头像 👤
4. 查看成员详情页
```

### 方式 2: 在线成员列表
```
1. 点击聊天室右上角 👥 图标
2. 查看在线成员列表
3. 点击任意成员
4. 查看成员详情页
```

1. 打开 City Chat 页面

### 核心代码
```dart
// 消息中的头像可点击
GestureDetector(
  onTap: () => _showUserDetail(message),
  child: Hero(
    tag: 'message_avatar_${message.userId}_${timestamp}',
    child: CircleAvatar(...),
  ),
```
