# 城市详情页功能说明

## 功能概述
实现了基于 Nomads.com 设计风格的城市详情页面，用户可以通过双击城市卡片进入详细信息页面。

## 使用方法

### 1. 进入数据服务页面
- 从主页点击"数据服务"快捷入口
- 或通过路由 `/data-service` 访问

### 2. 查看城市卡片
- 单击卡片：显示/隐藏快速评分信息覆盖层
- 双击卡片：跳转到城市详情页面

### 3. 城市详情页面内容

#### 顶部区域
- **大图展示**：城市代表性图片作为背景
- **排名徽章**：显示城市在榜单中的排名（如 #1 Best Place）
- **城市名称**：大标题显示城市名称
- **位置信息**：国家/地区图标和名称
- **操作按钮**：
  - 返回按钮（左上角）
  - 收藏按钮（右上角，点击切换收藏状态）
  - 分享按钮（右上角）

#### 基本信息区
- **天气标签**：当前温度和天气状况
- **网速标签**：平均网速（Mbps）
- **价格信息**：
  - 大字体显示月租金价格
  - "FOR A NOMAD" 标签说明

#### 评分卡片网格
显示 5 个关键指标，每个卡片包含：
- **图标**：表情符号图标
- **标签**：Overall、Cost、Internet、Liked、Safety
- **分数**：0-10 分制，带颜色编码
  - Overall：橙色 (🌟)
  - Cost：绿色 (💰)
  - Internet：蓝色 (📡)
  - Liked：红色 (👍)
  - Safety：紫色 (🛡️)

#### 关于城市
简要介绍城市的特点、优势和适合数字游民的原因。

#### 生活成本明细
列表展示各项生活开支：
- 一居室公寓租金
- 联合办公空间
- 餐厅用餐
- 咖啡
- 月度交通
- 健身房会员

#### 天气图表
12 个月的温度柱状图，展示全年气候变化趋势。

#### 照片画廊
网格布局展示城市多张照片（2列移动端，3列桌面端）。

## 响应式设计

### 移动端（< 768px）
- 顶部大图高度：300px
- 评分卡片：2 列网格
- 照片画廊：2 列网格
- 字体和间距适配小屏幕

### 桌面端（≥ 768px）
- 顶部大图高度：500px
- 评分卡片：5 列网格（一行显示全部）
- 照片画廊：3 列网格
- 更大的内边距和字体

## 技术实现

### 文件结构
```
lib/
├── pages/
│   ├── data_service_page.dart    # 城市列表页（双击触发跳转）
│   └── city_detail_page.dart     # 城市详情页
├── routes/
│   └── app_routes.dart            # 路由配置
└── controllers/
    └── data_service_controller.dart # 数据管理
```

### 路由配置
```dart
// 路由定义
static const String cityDetail = '/city-detail';

// 路由跳转（传递城市数据）
Get.toNamed(
  AppRoutes.cityDetail,
  arguments: cityData,
);
```

### 手势识别
```dart
GestureDetector(
  onTap: () => setState(() => showDetails = !showDetails),  // 单击
  onDoubleTap: () {                                          // 双击
    Get.toNamed(AppRoutes.cityDetail, arguments: widget.data);
  },
  child: ...,
)
```

## 设计参考
基于 Nomads.com 的设计风格：
- 深色渐变背景（#1a1a2e → #16213e）
- 白色卡片设计，带细边框和阴影
- 彩色评分指标
- 清晰的信息层级
- 现代简洁的排版
