# City Detail 分享功能使用指南

## 🎯 快速开始

### 用户如何分享信息

1. **打开城市详情页** - 浏览任意城市
2. **选择相关 Tab** - 切换到你想分享内容的标签页
3. **点击 + 按钮** - 点击右下角的红色浮动按钮
4. **查看对话框** - 阅读分享提示和说明
5. **开始分享** - 点击主按钮开始分享（当前显示 Coming Soon）

## 📱 各Tab分享功能说明

### Scores - 评分你的体验
- 📍 **用途**: 为城市的各项指标打分
- 💡 **提示**: 包括安全、生活质量、夜生活等15+项指标
- 🎯 **帮助**: 帮助其他数字游民了解城市的真实状况

### Guide - 分享生活技巧  
- 📍 **用途**: 分享实用的生活建议和技巧
- 💡 **提示**: 例如签证办理、交通出行、住宿推荐等
- 🎯 **帮助**: 帮助新来的游民快速适应城市

### Pros & Cons - 说出你的真实想法
- 📍 **用途**: 分享城市的优点和缺点
- 💡 **提示**: 投票支持你认同的观点
- 🎯 **帮助**: 帮助大家全面了解城市的好与坏

### Reviews - 撰写详细评论
- 📍 **用途**: 写一篇完整的城市体验评论
- 💡 **提示**: 可以上传照片，分享停留时长
- 🎯 **帮助**: 为其他人提供第一手的详细体验

### Cost - 分享真实费用
- 📍 **用途**: 分享你在城市的实际生活成本
- 💡 **提示**: 包括住宿、饮食、交通、健身等费用
- 🎯 **帮助**: 帮助其他人规划预算

### Photos - 上传美丽照片
- 📍 **用途**: 上传你拍摄的城市照片
- 💡 **提示**: 展示城市的风景、美食、文化等
- 🎯 **帮助**: 让其他人直观感受城市魅力

### Neighborhoods - 推荐好社区
- 📍 **用途**: 分享关于不同社区的信息
- 💡 **提示**: 包括安全程度、租金水平、氛围特点
- 🎯 **帮助**: 帮助新人选择合适的居住区域

## 🎨 UI 元素说明

### 浮动按钮
- **颜色**: 红色 (#FF4458) - 品牌主色调
- **图标**: 白色加号 (+)
- **位置**: 每个tab右下角固定位置
- **大小**: 标准FAB尺寸 (56x56)

### 分享对话框
- **顶部图标**: 与分享类型相关的大图标
- **标题**: 清晰的分享功能说明
- **描述**: 灰色辅助文字，说明如何贡献
- **按钮**: 全宽红色主按钮

## 💡 开发者注意事项

### 修改分享对话框
所有对话框方法位于 `city_detail_page.dart` 文件末尾：
- `_showShareScoreDialog()`
- `_showShareGuideDialog()`  
- `_showShareProsConsDialog()`
- `_showShareReviewDialog()`
- `_showShareCostDialog()`
- `_showSharePhotoDialog()`
- `_showShareNeighborhoodDialog()`

### 添加真实功能
将 `Get.snackbar(...)` 替换为实际的页面跳转或表单提交：

```dart
// 当前（Coming Soon）
onPressed: () {
  Get.back();
  Get.snackbar('Coming Soon', '...');
}

// 替换为真实功能
onPressed: () {
  Get.back();
  Get.to(() => ShareScoreFormPage(cityId: cityId));
}
```

## ✨ 设计理念

### 用户友好
- **一致性**: 所有tab的分享按钮位置、样式统一
- **可发现性**: 浮动按钮始终可见，不会被内容遮挡
- **清晰性**: 对话框明确说明需要分享什么内容

### 鼓励参与
- **降低门槛**: 一键即可开始分享流程
- **明确价值**: 告诉用户他们的分享如何帮助社区
- **即时反馈**: 点击后立即显示响应

### 未来扩展
- **易于修改**: 对话框方法独立，便于替换
- **统一风格**: 所有对话框使用相同的设计语言
- **灵活扩展**: 可轻松添加更多分享类型
