### 3. 使用 CustomScrollView 📜

**优化前**: `SingleChildScrollView`
**优化后**: `CustomScrollView` + `SliverToBoxAdapter` + `SliverPadding`

**优势**:
- ✅ 更好的滚动性能
- ✅ 与实际页面的 `NestedScrollView` 行为一致
- ✅ 支持复杂的滚动效果（如顶部固定、折叠等）
- ✅ 更流畅的骨架屏到实际内容的过渡

## 🔧 技术实现

### 1. 动态宽度设计
```dart
// 不同元素使用不同宽度，更自然
SkeletonBox(
  width: index == 0 ? 24 : 8,  // 轮播指示器
  width: index == 2 ? 200 : double.infinity,  // 内容行
)
```

### 2. 渐变蒙层
```dart
Container(
  decoration: BoxDecoration(
    gradient: LinearGradient(
      colors: [
        Colors.transparent,
        Colors.black.withValues(alpha: 0.6),
      ],
    ),
  ),
)
```

### 3. 阴影效果
```dart
BoxShadow(
  color: Colors.black.withValues(alpha: 0.08),
  blurRadius: 20,
  offset: const Offset(0, 4),
)
```

## 💡 设计原则

### 1. 内容优先
- 重要内容（大图、评分）使用更大的占位空间
- 次要内容（标签、按钮）使用较小的占位空间

### 2. 层次清晰
- 使用阴影和圆角区分不同层级的内容
- 保持与实际页面相同的视觉层次

### 3. 动态变化
- 不同元素使用不同的宽度和高度
- 避免所有元素都一样，更接近真实内容

### 4. 性能优先
- 使用 `CustomScrollView` 提升滚动性能
- 复用 shimmer 动画控制器
- 避免过度绘制
