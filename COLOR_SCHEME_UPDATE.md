# 配色方案更新 - 从深色系到浅灰色系

## 🎨 更新原因

用户反馈:**整体页面太暗,应该使用灰色或浅灰色作为主题色,而不是黑色**

原配色方案使用了大量黑色 (#212121),虽然极简但过于沉重,不够明亮舒适。

---

### Before (深色系 - 过暗)

|------|---------|------|

| 深色背景 | #424242 | 炭黑 - 太暗 ❌ |
| 卡片背景 | #212121, #424242, #546E7A | 深色系卡片 ❌ |

| 用途 | 颜色代码 | 说明 |
|------|---------|------|
| 主强调色 | #757575 | 中灰色 - 柔和 ✅ |

## 🔧 技术实现

### 批量替换技巧

使用 `sed` 命令批量替换颜色:
```bash
# 替换所有黑色边框为灰色
sed -i '' 's/Color(0xFF212121)/Color(0xFF757575)/g' file.dart

# 替换蓝色为灰色
sed -i '' 's/Color(0xFF1976D2)/Color(0xFF757575)/g' file.dart
```

### 颜色哈希分配

保持原有逻辑,根据名称哈希值分配颜色:
```dart
final colorIndex = name.hashCode.abs() % colorList.length;
final cardColor = colorList[colorIndex];
```
