## 📋 实现内容

### 1. 城市信息展示
当用户点击地图上的城市标记时：
- 📍 弹出底部信息卡片
- 🏙️ 显示城市名称和国家
- 👥 显示会员数量
- 🔘 提供两个操作按钮

### 2. 操作按钮
- **查看详情** - 跳转到城市详情页面
- **开始导航** - 打开地图应用选择器

### 3. 地图应用选择器
提供三个主流地图应用选择：
- 🔵 **谷歌地图** (Google Maps)
- 🟢 **高德地图** (Amap)
- 🟠 **百度地图** (Baidu Maps)

### 4. 智能启动逻辑
- ✅ 优先尝试打开原生 App
- ✅ App 未安装时自动使用 Web 版本
- ✅ 自动传递目的地名称和坐标
- ✅ 异常处理和错误提示

   - Baidu Maps URL Scheme
   - Web 备选方案
   - 异常处理

### URL Scheme 实现

| 地图应用 | URL Scheme | 支持平台 |
|---------|-----------|---------|
| Google Maps | `comgooglemaps://` | iOS + Android |
| 高德地图 (iOS) | `iosamap://` | iOS |
| 高德地图 (Android) | `androidamap://` | Android |
| 百度地图 | `baidumap://` | iOS + Android |

### Web 备选 URL

| 地图应用 | Web URL |
|---------|---------|
| Google Maps | `https://www.google.com/maps/` |
| 高德地图 | `https://uri.amap.com/` |
| 百度地图 | `https://api.map.baidu.com/` |
