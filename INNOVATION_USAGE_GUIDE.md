# Innovation 功能使用指南

## 快速开始

### 1. 创建创意项目

**步骤:**
1. 打开应用,进入"数据服务"页面
2. 点击 Innovation(创意项目)瓷片
3. 点击顶部的"创建我的创意项目"按钮
4. 填写项目信息:
   - 基本信息(项目名称、定位、封面)
   - 问题与解决方案
   - 市场定位
   - 竞争与商业模式
   - 当前进展与所需支持
   - 团队信息(可选)
5. 点击"发布创意项目"按钮

**提示:**
- 标有"必填"的字段必须填写
- 核心功能用逗号分隔,例如: "AI问答, 学习计划, 进度追踪"
- 建议上传一张吸引眼球的封面图

### 2. 浏览创意项目

**查看项目列表:**
- 在 Innovation 页面向下滚动
- 每个卡片显示:
  - 项目封面
  - 项目名称和定位
  - 产品类型标签
  - 核心功能标签
  - 创建者信息
  - 更新时间

**与项目互动:**
- 点击"查看详情"按钮 → 查看完整项目信息
- 点击"联系作者"按钮 → 与项目创建者一对一聊天

### 3. 查看项目详情

**详情页包含 13 个部分:**
1. 项目名称和封面
2. 一句话定位
3. 要解决的问题
4. 解决方案
5. 目标用户
6. 产品形态
7. 核心功能列表
8. 竞争优势
9. 商业模式
10. 市场潜力
11. 当前进展
12. 团队介绍
13. 所需支持

### 4. 联系项目创建者

**一对一聊天:**
1. 在列表页点击任意项目卡片的"联系作者"按钮
2. 进入聊天页面
3. 直接与项目创建者沟通

**适用场景:**
- 对项目感兴趣,想了解更多
- 有投资意向
- 想加入团队成为合伙人
- 提供技术支持
- 市场合作机会

## 示例项目

### 智课通 - AI 学习助手
- **定位**: 为大学生打造的 AI 驱动学习伴侣
- **目标用户**: 大学生
- **产品形态**: 移动应用 + 网站
- **核心功能**: AI 问答、学习计划、进度追踪

### 碳迹追踪 - 个人碳足迹追踪
- **定位**: 帮助个人追踪和降低碳排放的移动应用
- **目标用户**: 环保意识强的年轻人
- **产品形态**: 移动应用
- **核心功能**: 碳足迹计算、减排建议、社区挑战

### 灵犀翻译 - 专业文档翻译
- **定位**: 面向专业领域的高质量 AI 翻译平台
- **目标用户**: 企业、学者、专业译者
- **产品形态**: 网站 + API
- **核心功能**: 多语言翻译、术语管理、质量保证

## 导航路径

```
主页
 └─ 数据服务页
     └─ Innovation 瓷片
         ├─ 创建我的创意项目 → 新建页面
         └─ 探索创意项目
             ├─ 查看详情 → 详情页
             └─ 联系作者 → 聊天页
```

## 常见问题

**Q: 如何修改已发布的项目?**
A: 目前暂不支持编辑功能,即将推出。

**Q: 可以删除项目吗?**
A: 目前暂不支持删除功能,即将推出。

**Q: 项目封面图有大小限制吗?**
A: 建议使用 1920x1080 尺寸,系统会自动压缩。

**Q: 核心功能最多可以填几个?**
A: 没有限制,但建议 3-5 个最核心的功能。

**Q: 联系作者后对方能看到我的信息吗?**
A: 是的,聊天功能会显示你的用户名和头像。
