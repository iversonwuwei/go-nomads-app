# 登录页面 - Material Design 3 风格重构

## 🎨 设计理念

采用 Google Material Design 3 (Material You) 设计语言，打造现代化、简洁、美观的移动端登录体验。

## ✨ 核心特性

### 1. **视觉设计**
- **纯白背景**：简洁干净的白色背景，符合现代审美
- **品牌色**：`#1976D2` Material Blue 作为主题色
- **圆角设计**：统一使用 `12.r` 圆角，柔和舒适
- **阴影效果**：Logo 区域使用品牌色阴影，增强层次感

### 2. **品牌标识**
- **居中 Logo**：64x64 蓝色圆角方形 Logo
- **品牌阴影**：蓝色光晕效果，突出品牌特征
- **清晰标题**：大字号欢迎文字（32sp），负字间距优化

### 3. **登录方式切换**
- **分离式标签**：两个独立的圆角按钮
- **选中状态**：品牌蓝色填充 + 白色文字
- **未选中状态**：灰色背景 + 黑色文字
- **平滑过渡**：InkWell 水波纹点击效果

### 4. **输入框设计**
- **Material 3 样式**：
  - 填充背景：`Colors.grey[50]`
  - 边框圆角：12
  - 聚焦边框：2px 品牌蓝
  - 普通边框：1px 浅灰
  - 错误边框：红色提示
- **图标前缀**：
  - 手机号：`Icons.phone_android`
  - 密码：`Icons.lock_outline`
  - 验证码：`Icons.sms_outlined`
- **浮动标签**：`labelText` 自动浮起效果
- **统一高度**：52h 舒适触控高度

### 5. **密码输入框**
- **可见性切换**：眼睛图标切换显示/隐藏
- **图标位置**：右侧 suffix icon
- **状态响应**：Obx 响应式更新

### 6. **验证码输入**
- **组合布局**：输入框 + 获取按钮
- **按钮样式**：
  - 白色背景 + 蓝色边框
  - 禁用状态：灰色背景 + 文字
  - 宽度：110w，高度：52h
- **倒计时显示**：60s 倒计时，动态更新

### 7. **登录按钮**
- **Material 3 规范**：
  - 品牌蓝填充背景
  - 白色文字
  - 52h 高度
  - 12r 圆角
  - 无阴影（elevation: 0）
- **加载状态**：白色圆形进度条
- **文字设计**：17sp，字重 600，字间距 0.5

### 8. **第三方登录**
- **图标按钮设计**：
  - 52x52 方形按钮
  - 12r 圆角
  - 彩色图标 + 淡色背景
  - 边框：对应颜色 20% 透明度
- **支持平台**：
  - 微信：`#07C160` 绿色
  - Apple：`#000000` 黑色
  - Facebook：`#1877F2` 蓝色
- **布局方式**：居中横向排列，间距 24w

### 9. **文字层级**
- **标题**：32sp，粗体，负字间距 -0.5
- **副标题**：16sp，中等透明度
- **输入框**：16sp，标准字重
- **按钮**：17sp，半粗体
- **提示文字**：14-15sp

## 📱 移动端布局

### 结构层次
```
SingleChildScrollView
└── Padding (24w horizontal)
    └── Column
        ├── Logo (64x64, 圆角阴影)
        ├── 欢迎文字 (32sp 粗体)
        ├── 登录方式切换 (分离式标签)
        ├── 表单输入区域
        ├── 登录按钮 (52h)
        ├── 分隔线 ("或使用其他方式登录")
        ├── 第三方登录图标
        └── 注册提示
```

### 间距规范
- 顶部到 Logo：60h
- Logo 到标题：32h
- 标题到副标题：8h
- 副标题到登录切换：48h
- 切换到表单：16h
- 输入框间距：16-20h
- 表单到按钮：16-24h
- 分隔线间距：24h
- 第三方登录：24h
- 底部留白：40h

## 🎯 设计优势

### 1. **无文字变形**
- 使用 `.sp` 响应式单位
- 固定字体大小，保证可读性
- 合理的行高和字间距

### 2. **符合触控规范**
- 按钮最小高度 52h
- 图标大小 22-28sp
- 足够的点击区域

### 3. **视觉层次清晰**
- Logo → 标题 → 表单 → 操作
- 颜色对比度符合 WCAG 标准
- 重要元素视觉突出

### 4. **交互反馈完善**
- InkWell 水波纹效果
- 输入框聚焦状态
- 按钮禁用状态
- 加载动画

### 5. **品牌一致性**
- 统一品牌蓝 #1976D2
- 统一圆角 12r
- 统一间距规范

## 🔄 响应式设计

### 桌面端 (>800px)
- 保持原有左右分栏布局
- 左侧：品牌展示区
- 右侧：登录卡片

### 移动端 (≤800px)
- 单列垂直布局
- 顶部 Logo 居中
- 内容区域可滚动
- 底部留白舒适

## 🚀 技术实现

### 响应式单位
- `.w` - 宽度响应式
- `.h` - 高度响应式
- `.sp` - 字体响应式
- `.r` - 圆角响应式

### 状态管理
- GetX Obx() 响应式
- 表单验证器
- 加载状态控制
- 倒计时逻辑

### Material 组件
- TextFormField (Material 3)
- ElevatedButton
- InkWell 水波纹
- Checkbox 圆角样式
