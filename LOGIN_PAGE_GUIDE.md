# 新版登录页面使用指南

## 📱 页面特性

### 🎨 视觉设计
- **深色渐变背景**：采用从深蓝到蓝绿的渐变色（#0F2027 → #203A43 → #2C5364）
- **左右分栏布局**：桌面端（宽度>800px）显示左右分栏，移动端自适应单栏
- **白色登录卡片**：右侧白色卡片带有阴影和圆角，提供良好的视觉层次感

### 🚀 功能模块

#### 左侧品牌区域（仅桌面端显示）
1. **品牌Logo和标题**
   - API Logo（白色方形背景）
   - 主标题："数金 API 管理平台"
   - 副标题："提供企业级数字化开发者统一入口"

2. **宣传口号**
   - "一站式开放平台"
   - "驱动数字金融创新"

3. **产品描述**
   - 详细说明平台核心能力和服务特色

4. **核心特性列表**
   - 📡 统一 API / IP、标准化接口文档及接口工具
   - 🔐 企业级权限管理与身份安全认证
   - ⚡ 实时接口调用监控版，精准提高价格

#### 右侧登录卡片
1. **欢迎标题**
   - "欢迎回来"
   - 使用说明文字

2. **登录方式切换**
   - 手机号登录（密码登录）
   - 验证码登录
   - 灰色背景标签页，选中项白色高亮

3. **表单输入**
   - **账号输入框**：支持邮箱/手机号/用户名
   - **密码输入框**：带显示/隐藏密码按钮
   - **验证码输入框**：带获取验证码按钮（60秒倒计时）

4. **辅助功能**
   - 记住我：复选框
   - 忘记密码：蓝色链接（仅密码登录时显示）

5. **操作按钮**
   - **立即登录**：蓝色主按钮（#1890FF）
   - **试用申请**：灰色文字链接

6. **第三方登录**
   - 微信登录：绿色图标（#07C160）
   - 支付宝登录：蓝色图标（#1677FF）

7. **注册入口**
   - "还没有账号？立即注册"链接

8. **客服信息**
   - 底部显示客服服务热线

## 🎯 颜色规范

### 主色调
- **主题蓝色**：#1890FF（按钮、链接、边框激活状态）
- **深色渐变**：
  - #0F2027（起始）
  - #203A43（中间）
  - #2C5364（结束）

### 辅助色
- **微信绿**：#07C160
- **支付宝蓝**：#1677FF
- **文字灰**：#666666 (Grey[600])
- **边框灰**：#E5E5E5 (Grey[200])
- **背景灰**：#F5F5F5
- **输入框背景**：#FAFAFA

## 📐 布局规范

### 桌面端（宽度 > 800px）
- 左右各占 50% 宽度
- 登录卡片最大宽度：500px
- 卡片内边距：40px
- 左侧内容区内边距：60px

### 移动端（宽度 ≤ 800px）
- 单栏布局
- 整体内边距：20px
- 顶部留白：40px

## 🔧 技术实现

### 响应式设计
```dart
final screenWidth = MediaQuery.of(context).size.width;
final isLargeScreen = screenWidth > 800;
```

### 渐变背景
```dart
decoration: const BoxDecoration(
  gradient: LinearGradient(
    begin: Alignment.topLeft,
    end: Alignment.bottomRight,
    colors: [
      Color(0xFF0F2027),
      Color(0xFF203A43),
      Color(0xFF2C5364),
    ],
  ),
)
```

### 登录方式状态管理
- 使用GetX的响应式编程
- `LoginType.phonePassword`：密码登录
- `LoginType.phoneCode`：验证码登录

## 📝 表单验证

### 手机号验证
- 仅允许数字输入
- 最大长度限制：11位
- 使用 `FilteringTextInputFormatter.digitsOnly`

### 密码验证
- 支持密码可见性切换
- 使用 `controller.passwordValidator`

### 验证码验证
- 6位数字验证码
- 60秒发送间隔限制
- 倒计时显示

## 🎭 交互效果

1. **输入框聚焦**
   - 未激活：灰色边框（Grey[200]）
   - 激活：蓝色边框（#1890FF）
   - 填充背景：#FAFAFA

2. **按钮状态**
   - 正常：蓝色背景
   - 禁用：灰色背景
   - 加载中：显示圆形进度指示器

3. **标签页切换**
   - 选中：白色背景 + 阴影效果
   - 未选中：透明背景

## 📱 移动端适配

- 自动切换为单栏布局
- 左侧品牌区域在移动端隐藏
- 仅显示登录卡片
- 所有尺寸使用ScreenUtil适配
