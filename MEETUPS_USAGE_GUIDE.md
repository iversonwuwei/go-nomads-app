# Meetups 功能快速使用指南

## 🚀 如何使用

### 1. 从首页访问 Meetups

1. 打开应用,进入首页
2. 在快捷功能区找到 **Meetups** 图标 (👥 图标)
3. 点击进入 Meetups 列表页

### 2. 浏览 Meetups

#### 切换标签
- **All Meetups**: 查看所有即将举行的活动
- **Joined**: 查看已加入的活动
- **Past**: 查看已结束的历史活动

#### 筛选活动
- 点击右上角 **筛选图标** (🔽)
- 选择活动类型:
  - All Types (所有类型)
  - Coffee (咖啡聚会)
  - Coworking (共同办公)
  - Activity (户外活动)
  - Language (语言交流)

### 3. 查看活动详情

1. 在列表中点击任意活动卡片
2. 进入详情页查看:
   - 📸 活动图片
   - 📅 日期和时间
   - 📍 场地和地址
   - 👥 参与人数和剩余名额
   - 📝 活动描述
   - 👤 组织者信息
   - 👫 参与者列表

### 4. 加入活动

#### 在列表页:
- 点击活动卡片右下角的 **Join** 按钮
- 按钮变为 **Joined** 表示已成功加入

#### 在详情页:
- 点击底部的 **Join Meetup** 按钮
- 加入后按钮变为 **Leave Meetup**
- 可随时点击退出活动

### 5. 聊天交流

#### 活动群聊:
- **列表页**: 点击卡片右下角的 **💬 聊天图标**
- **详情页**: 点击底部的 **Chat** 按钮

#### 私信组织者:
- 在详情页组织者信息卡片
- 点击 **Message** 按钮

### 6. 创建新活动

1. 在 Meetups 列表页
2. 点击右上角的 **➕ 创建图标**
3. 填写活动信息 (跳转到创建页面)

### 7. 分享活动

- 在详情页点击右上角的 **分享图标** (🔗)
- 可分享活动给其他用户

## 📱 UI 元素说明

### 活动类型标签
- 🟤 **Coffee** - 咖啡聚会
- 🔵 **Coworking** - 共同办公
- 🟢 **Activity** - 户外活动
- 🟣 **Language** - 语言交流

### 状态提示
- 🔴 **Starting Soon** - 24小时内开始
- 🟠 **Full** - 已满员,无法加入
- ⚫ **Ended** - 活动已结束

### 按钮状态
- **Join** (红色) - 点击加入活动
- **Joined** (浅红色) - 已加入,点击可退出
- **Leave Meetup** (灰色) - 退出活动
- **Full** (橙色) - 已满员,无法操作
- **Ended** (灰色) - 已结束,无法操作

## 🎯 常见场景

### 场景 1: 新用户想找活动
1. 打开首页 → 点击 Meetups
2. 浏览 "All Meetups" 标签
3. 选择感兴趣的活动 → 查看详情
4. 点击 "Join Meetup" 加入

### 场景 2: 查看已加入的活动
1. 进入 Meetups 列表页
2. 切换到 "Joined" 标签
3. 查看所有已加入的活动
4. 点击进入详情或聊天

### 场景 3: 联系组织者
1. 进入活动详情页
2. 滚动到组织者信息区域
3. 点击 "Message" 按钮
4. 发送私信咨询

### 场景 4: 退出活动
1. 在 "Joined" 标签找到活动
2. 进入详情页
3. 点击底部 "Leave Meetup" 按钮
4. 确认退出

## 💡 提示

- ⏰ 活动即将开始(24小时内)会有 "Starting Soon" 提示
- 👥 参与人数达到上限时无法加入
- 📅 已结束的活动可在 "Past" 标签查看
- 💬 随时可以通过聊天功能与其他参与者交流
- 🔄 下拉列表页可刷新数据
