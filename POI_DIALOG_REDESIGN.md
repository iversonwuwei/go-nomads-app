# POI 信息对话框重新设计

## 设计目标

将原有的简单 AlertDialog 重新设计为现代化、视觉吸引力强的自定义对话框。

## 设计改进

### 1. 视觉层次 ✨

#### 之前
- 简单的 AlertDialog
- 信息平铺展示
- 缺乏视觉重点

#### 现在
- 自定义 Dialog 容器
- 清晰的视觉层次
- 渐变色彩顶部区域
- 卡片式信息展示

### 2. 顶部设计 🎨

**特点**：
- **渐变背景**: 使用 POI 类型对应的主题色渐变
- **大图标**: 白色半透明圆角容器 + 32px 图标
- **类型标签**: 白色半透明胶囊标签显示 POI 类型
- **大标题**: 20px 粗体白色文字显示 POI 名称

**代码实现**：
```dart
Container(
  decoration: BoxDecoration(
    gradient: LinearGradient(
      colors: [
        _getPOIColor(poi.type),
        _getPOIColor(poi.type).withValues(alpha: 0.8),
      ],
    ),
  ),
)
```

### 3. 信息展示 📊

#### 距离信息卡片
- 浅灰色背景 (Colors.grey[50])
- 细边框 (Colors.grey[200])
- 16px 圆角
- 左侧彩色图标 (near_me)
- 两行文本：
  - 上：灰色小字 "距离 XXX"
  - 下：20px 粗体黑色距离数值

#### 坐标信息卡片
- 相同的卡片样式
- 两行坐标信息：
  - 经度
  - 纬度
- 中间细分割线
- 每行包含：图标 + 标签 + 数值

#### 提示信息
- 蓝色浅背景 (Colors.blue[50])
- 信息图标 + 提示文本
- 引导用户交互

### 4. 底部操作 🎯

**两个按钮**：

1. **关闭按钮** (OutlinedButton)
   - 占 1 份宽度
   - 灰色边框
   - 次要视觉权重

2. **在地图上查看** (ElevatedButton)
   - 占 2 份宽度（更突出）
   - POI 主题色背景
   - 带图标 (my_location)
   - 主要操作
   - 点击后聚焦到 POI 位置 (缩放 17)

### 5. 动画和交互 🎬

- **背景遮罩**: 50% 黑色半透明
- **圆角**: 24px 大圆角，现代感十足
- **阴影**: 模糊 20px，向下偏移 10px
- **最大宽度**: 400px (适配各种屏幕)
- **内边距**: 24px 统一内边距

## 信息架构

### 显示的信息

1. **POI 类型** (顶部标签)
2. **POI 名称** (顶部大标题)
3. **距离** (第一个卡片)
   - 动态计算距离 Coworking Space
   - 小于 1000m 显示米，否则显示公里
4. **经度** (第二个卡片)
5. **纬度** (第二个卡片)
6. **使用提示** (底部提示卡片)

### 新增功能

#### `_buildInfoRow` 方法
构建坐标信息行的辅助方法：

```dart
Widget _buildInfoRow({
  required IconData icon,
  required String label,
  required String value,
  required Color color,
})
```

**参数**：
- `icon`: 左侧图标
- `label`: 信息标签 (如 "经度")
- `value`: 信息数值
- `color`: 图标颜色 (使用 POI 主题色)

#### `_focusOnLocation` 方法
聚焦到指定位置：

```dart
void _focusOnLocation(LatLng position) {
  _mapController.move(position, 17.0);
}
```

**功能**：
- 移动地图中心到指定位置
- 缩放到 17 级 (详细视图)
- 用于"在地图上查看"按钮

## 颜色系统

### POI 类型颜色
根据 POI 类型动态显示不同主题色：

- **交通** (Transit): 蓝色
- **住宿** (Accommodation): 紫色
- **餐饮** (Restaurant): 橙色

### 应用位置
1. 顶部渐变背景
2. 类型图标容器背景
3. 距离卡片图标背景
4. 坐标信息图标颜色
5. "在地图上查看"按钮背景

## 设计细节

### 圆角统一
- 对话框外层: 24px
- 顶部区域: 24px (仅上方)
- 卡片: 16px
- 按钮: 12px
- 图标容器: 12-16px
- 标签: 12px

### 间距统一
- 外边距: 24px
- 卡片间距: 16px
- 卡片内边距: 16px
- 小间距: 8px, 12px
- 按钮间距: 12px

### 字体层级
- 主标题: 20px, Bold
- 距离数值: 20px, Bold
- 按钮文字: 16px, SemiBold
- 标签文字: 12-13px, Regular/SemiBold
- 提示文字: 12px

## 技术实现

### 容器层级
```
Dialog
└── Container (白色，24px圆角，阴影)
    ├── 顶部区域 (渐变背景)
    │   ├── 图标容器
    │   └── 文字区域
    │       ├── 类型标签
    │       └── 名称标题
    ├── 信息区域 (24px内边距)
    │   ├── 距离卡片
    │   ├── 坐标卡片
    │   └── 提示卡片
    └── 底部区域 (24px内边距)
        ├── 关闭按钮
        └── 查看按钮
```

### 响应式设计
- 使用 `constraints: BoxConstraints(maxWidth: 400)`
- 水平内边距 24px
- 文字自适应换行
- 弹性布局 (Expanded)

## 设计灵感

### 参考元素
- Material Design 3.0 卡片设计
- iOS 地图标注弹窗
- 现代化 App 信息展示卡片
- 渐变色顶部设计趋势

### 设计原则
1. **对比度**: 白色背景 + 彩色顶部
2. **一致性**: 统一的圆角、间距、字体
3. **层级**: 卡片、阴影、颜色建立视觉层级
4. **引导**: 主次按钮明确引导用户操作
5. **美观**: 渐变、圆角、阴影创造现代感
