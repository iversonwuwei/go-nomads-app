# Flutter + GetX 购物应用功能说明

## 应用概述
这是一个基于Flutter和GetX状态管理的现代化购物应用，支持用户认证、商品浏览、购物功能等。

## 核心功能

### 1. 应用架构
- **状态管理**: GetX状态管理库
- **路由管理**: GetX路由系统
- **项目结构**: MVC模式，分离控制器、页面、模型和路由

### 2. 主要页面

#### 主页 (HomePage)
- 轮播图横幅
- 快捷功能入口（扫码、搜索、消息、客服）
- 热门商品网格展示
- 商品图片缓存优化

#### 个人中心 (ProfilePage)
- 用户信息展示
- 功能菜单（订单、收藏、购物车等）
- 设置选项
- 登录/注册入口

#### 登录页 (LoginPage)
- **多种登录方式**:
  - 手机号+密码登录
  - 手机号+验证码登录
  - 第三方登录（微信、支付宝）
- **完整认证流程**:
  - 用户登录
  - 用户注册
  - 密码重置
- **表单验证**: 实时输入验证和错误提示

### 3. 状态管理

#### 购物控制器 (ShoppingController)
- 轮播图数据管理
- 商品数据管理
- 商品加载状态控制

#### 认证控制器 (AuthController)
- 用户认证状态管理
- 登录类型切换（密码/验证码）
- 认证模式切换（登录/注册/重置密码）
- 表单验证逻辑
- 第三方登录集成

### 4. 数据模型

#### 商品模型 (ProductModel)
- 商品基本信息
- 价格、图片、描述等

#### 横幅模型 (BannerModel)
- 横幅图片和链接
- 轮播图数据结构

### 5. UI特性
- **Material Design 3**: 现代化UI设计
- **响应式布局**: 适配不同屏幕尺寸
- **优雅动画**: 平滑的页面切换和交互动画
- **主题定制**: 一致的色彩方案和样式

## 导航流程

### 应用启动
1. 启动应用 → 主页 (MainPage)
2. 底部导航：首页 ↔ 个人中心

### 认证流程
1. 个人中心 → 点击"登录/注册" → 登录页
2. 登录页提供三种认证模式：
   - 登录 (Login)
   - 注册 (Register)  
   - 忘记密码 (ForgotPassword)

### 登录方式
- **密码登录**: 手机号 + 密码
- **验证码登录**: 手机号 + 短信验证码
- **第三方登录**: 微信、支付宝一键登录

## 技术特点
- **响应式状态管理**: 使用GetX Rx变量实现响应式更新
- **依赖注入**: 控制器自动注册和依赖管理
- **路由管理**: 命名路由和页面导航
- **表单验证**: 实时验证和错误处理
- **UI组件化**: 可复用的UI组件设计

这个应用为进一步开发更复杂的电商功能提供了坚实的基础架构。
