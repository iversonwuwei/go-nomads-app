# 贪吃蛇游戏功能说明

## 🐍 功能概述

已成功为您的数金数据应用添加了贪吃蛇游戏功能，包含15秒无操作自动进入游戏的特性。

## ✨ 主要功能

### 1. 自动触发机制
- **触发条件**：在AI聊天页面超过15秒无任何操作
- **倒计时提醒**：最后5秒会显示倒计时警告
- **自动跳转**：时间到达后自动跳转到贪吃蛇游戏

### 2. 操作重置
- 发送消息会重置15秒计时器
- 点击快捷功能按钮会重置计时器
- 任何用户交互都会重新开始计时

### 3. 贪吃蛇游戏特性
- **游戏区域**：20x30网格的游戏场地
- **控制方式**：屏幕下方的方向键控制
- **游戏状态**：准备、游戏中、暂停、游戏结束
- **分数系统**：吃到食物+10分
- **碰撞检测**：撞墙或撞到自己身体会游戏结束

## 🎮 游戏界面

### 顶部信息栏
- 当前分数显示
- 蛇身长度显示
- 返回按钮

### 游戏区域
- 深色主题设计
- 绿色蛇身，红色食物
- 网格边框设计

### 控制区域
- 开始/暂停/重置按钮
- 四方向控制键
- 游戏状态提示

### 关键功能
1. **Timer机制**：使用Dart的Timer实现15秒倒计时
2. **GetX状态管理**：响应式状态更新
3. **游戏循环**：300毫秒间隔的游戏刷新
4. **碰撞检测**：墙壁和自身碰撞检测
5. **随机食物生成**：避免与蛇身重叠

## 🎯 使用方法

1. **进入AI聊天页面**
2. **等待15秒不进行任何操作**
3. **观察倒计时提醒（最后5秒）**
4. **自动跳转到贪吃蛇游戏**
5. **点击开始按钮开始游戏**
6. **使用方向键控制蛇的移动**
