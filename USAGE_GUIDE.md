# Flutter + GetX 购物应用使用指南

## 快速体验登录功能

### 1. 启动应用
应用已经在模拟器中运行，你可以看到：
- 现代化的购物首页
- 底部导航栏（首页 / 个人中心）

### 2. 访问登录页面
**方法1：通过个人中心**
1. 点击底部导航的"个人中心"标签
2. 在设置菜单中找到"登录/注册"选项
3. 点击进入登录页面

### 3. 登录页面功能展示

#### 登录方式切换
- **密码登录**: 手机号 + 密码
- **验证码登录**: 手机号 + 短信验证码
- 页面顶部可以切换登录方式

#### 认证模式
- **登录**: 用户登录
- **注册**: 新用户注册  
- **忘记密码**: 密码重置
- 页面顶部Tab可以切换模式

#### 第三方登录
- 微信登录按钮
- 支付宝登录按钮
- 一键登录体验

### 4. 表单验证体验
尝试以下操作来体验表单验证：

**手机号验证**
- 输入空白或格式错误的手机号
- 查看实时错误提示

**密码验证**  
- 输入少于6位的密码
- 查看密码强度提示

**验证码验证**
- 切换到验证码登录
- 体验验证码倒计时功能

### 5. UI特性体验

#### 响应式设计
- 尝试旋转设备查看布局适应
- 体验不同屏幕尺寸的适配

#### 动画效果
- 页面切换的平滑动画
- 按钮点击的反馈动画
- Tab切换的流畅效果

#### 主题风格
- 一致的Material Design 3设计
- 优雅的色彩搭配
- 现代化的UI元素
