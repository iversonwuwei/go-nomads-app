## 常见问题

### Q1: 地图为什么是灰色占位符?
**A:** 这是 MVP 版本,使用占位符代替真实地图。未来会集成高德地图 SDK。

### Q2: 为什么只有 Bangkok 的数据?
**A:** 当前版本硬编码了 Bangkok 的 9 个示例 Venue。未来会从后端 API 获取动态数据。

### Q3: 标记点位置是真实的吗?
**A:** 不是。当前使用简单算法分布标记点。真实地图会使用经纬度精确定位。

### Q4: 如何添加更多 Venue?
**A:** 在 `VenueMapPickerPage._venues` 列表中添加新的 Map 对象即可。
